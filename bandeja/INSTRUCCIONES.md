# Bandeja de entrada — Manual de soporte

Escribe aquí lo que quieres agregar o actualizar en el manual de soporte. Puedes ser todo lo informal que quieras — Claude Code se encarga de redactarlo bien.

## Formato

Cada cosa que quieras agregar va como un bloque separado. Incluye:

1. **Qué agregar** — describe la información, procedimiento, o cambio
2. **Pantallazos** (opcional) — nombra los archivos que dejaste en esta carpeta y describe brevemente qué muestran

### Ejemplo

```
## Cómo exportar datos desde Admin

Agregar instrucciones para exportar la lista de usuarios desde Admin Circles.
Los pasos son: ir a Admin > Users > Export > seleccionar formato CSV.

Pantallazos:
- captura1.png → muestra el botón Export en la vista de usuarios
- captura2.png → muestra el diálogo de selección de formato
```
